6.041x - Unit 4: Discrete Random Variables¶

Notes by Leo Robinovitch¶

Core Concepts:¶

Random Variable: a numerical quantity whose value is determined by the outcome of the probabilistic experiment (e.g. the weight of a randomly selected person)

  • Abstract concept of weight: $\mathbf{W}$ (random variable)

  • Particular weight of given student: $\mathbf{w}$ (actual number)

  • Weight is an object whose value is determined once the outcome of the prob. experiment is known

  • Can think of Weight ($W$) as black box/function: $student \to \mathbf{W(student)} \to w$

  • Possible for a given probabilistic experiment to have multiple associated random variables (e.g. weight W, height H, etc.)

    • Create new random variable based off other random variables, e.g. $BMI = B = \frac{W}{H^2}$ (still a function of outcomes)

  • Random variables are functions mapping $\Omega$ to real numbers (discrete or continuous values)

    • Meaning of $X + Y$: random variable takes value $x + y$, where X takes value x and Y takes value y given some outcome

  • Since some of the outcomes of a probabilistic experiment are more likely than others, the associated numerical values of random variables will be more likely than others

Probability Mass Functions (PMF, Probability Law or Probability Distribution): description of relative likelihoods for values of random variable

  • Define the y-axis of a PMF for random variable X, $p_X(x) = P(X=x) = P(\{\omega \in \Omega\ such\ that\ X(\omega)=x\})$

    • $PMF = f(x)$, where for any x, the PMF specifies the probability that the random variable X takes on this particular value x

  • Properties:

    • $p_X(x) \geq 0$ (like any probability)

    • $\sum\limits_{x}p_X(x)=1$

  • To calculate PMF:
    • Determine sample space and probability rule (e.g. 2 fair dice rolls $\to$ 4x4 grid where each outcome has probability $\frac{1}{16}$
    • Determine random variable (e.g. Z = sum of 2 dice rolls)
    • Calculate Z for all outcomes
    • Plot $p_Z(z)$ for all z


Discrete Random Variable Example 1: Bernoulli with Parameter $\mathbf{p \in [0,1]}$: models trial resulting in success/failure (e.g. coin, yes/no, 0 or 1 etc.)

  • X = 1 with probability $p$
  • X = 0 with probability $1 - p$
  • PMF is simply two bars: at x = 0, height $p$, and at x = 1 height $1-p$


Indicator Variables: Bernoulli random variables that allow us to translate manipulations of events into manipulations of random variables

  • E.g. sample space is split into $A$ and $A^c$.
    • For indicator variable $I_A \to P_{I_A}(1) = P(I_A = 1) = P(A)$


Discrete Random Variable Example 2: Uniform with Parameters $\mathbf{a,b \in \mathbb{Z}}$ (integers): a "setup" or "ignorant" model with equal probabilities for all random variable values

  • Here, the outcome of the experiment is the same as the value of the random variable $\to X(\omega) = \omega$
  • Model of complete igorance, e.g. looking at digital clock and guessing the second value (a = 0, b = 59)


Discrete Random Variable Example 3: Binomial with Parameters $\mathbf{n}$ and $\mathbf{p \in [0,1]}$: models experiment of $n$ trials with $P(success) = p$

  • E.g. $n$ tosses of coin with $P(heads) = p$
    • Sample space $\Omega$: set of sequences of H & T of length n $\to$ HH, HT, TH, TT for n = 2
    • Random variable X: number of heads observed
    • Models fixed number of identical and independant trials where trial outcomes are success or failure with probabilities $p$ and $1-p$ respectively

  • Y-Axis of PMF is same as previous binomial formula: $p_X(k) = \binom{n}{k}p^k(1-p)^{n-k}$ for $k = 0,1,...,n$


Discrete Random Variable Example 4: Geometric with Parameter $\mathbf{0 < p \leq 1}$: models experiment infinitely many independent success/failure trials with $P(success) = p$

  • Random variable $X$: number of trials until first success

  • E.g. flip a coin forever:

    • Sample space is set of infinite sequences of H & T
    • Random variable $X$: number of flips until first heads

  • If k is number of trials until specified event: $p_X(k) = P(X = k) = P(FFF...FS) = P(F)^{k-1}P(S) = (1-p)^{k-1}p,\ where\ k = 1, 2, 3...$

  • Used to model waiting times, number of trials until an event, etc.
  • Definitely related to survival analysis


Expectation: one form of summary of a random variable (the average, mean, etc.) $\to$ the average outcome in a large number of independent trials

  • E.g.: X is random variable of "gain" in a trial. $p_X(1) = \frac{2}{10}, \ \ p_X(2) = \frac{5}{10}, \ \ p_X(3) = \frac{3}{10}$
    • Can say the expected (average) gain is $\frac{1*200 + 2*500 + 3*300}{1000} = 1p_X(1) + 2p_X(2) + 3p_X(3)$

  • Generally, $\mathbf{E[X] = \sum\limits_{x}xp_X(x)}$
  • Assumption: for infinite sums, $E[X] = \sum\limits_{x}|x|p_X(x)<\infty$ (infinite sum well defined)
  • Average of a population: e.g. n students, weight of i'th student is $x_i$
    • If all equally likely to be chosen and $x_i$'s are distinct: $E[X] = \sum\limits_{i}x_i\frac{1}{n} = \frac{1}{n}\sum\limits_{i}x_i$
  • Properties:
    • If for all $\omega, \ X(\omega) \geq 0$, that is, $X \geq 0$, then $E[X] \geq 0$
    • If $a \leq X \leq b$, then $a \leq E[X] \leq b$
    • For a constant c, $E[c] = c$

Expectation of Bernoulli Random Variable:

  • Recall $p_X(1) = p$ and $p_X(0) = 1-p$

  • As such, $E[x] = 1p + 0(1-p) = p \to \mathbf{E[X] = p}$
    • If $X$ is indicator variable $X = I_A$ (i.e. $X = 1$ if A occurs, $X=0$ otherwise, such that $p = P(A)$) $\to E[I_A] = P(A)$

Expectation of Uniform Random Variable:

  • E.g. uniform on [0,n] (a = 0, b = n): $p_X(x) = \frac{1}{n+1}$
    • $E[X] = \frac{1}{n+1}(0 + 1 + 2 + ... + n) = \frac{1}{n+1}\frac{n(n+1)}{2} = \frac{n}{2}$

  • Generally, $E[X] = \frac{a+b}{2} \to$ the midpoint! AKA the "balance point"

Expected Vale Rule for Calculating $\mathbf{E[g(X)]}$:

  • Let $X$ be a random variable and $Y = g(X) \to$ 2 ways to calculate $E[Y]$:
    1. Averaging over y: $E[Y] = \sum\limits_{y}yp_Y(y)$
      • $E[Y] = 3*(0.1 + 0.2) + 4 * (0.3 + 0.4)$

    2. Averaging over x (Expected Value Rule): $\mathbf{E[Y] = E[g(X)] = \sum\limits_{x}g(x)p_X(x)}$
      • $E[Y] = 3*0.1 + 3*0.2 + 4*0.3 + *0.4$

  • Number 2 has the advantage that we don't have to find the pmf of new variable ($p_Y(y)$ not needed)
  • Application:
    • $E[X^2] = \sum\limits_{x}x^2p_X(x) \to$ here, $g(x) = x^2$
    • Note that $E[g(x)] \neq g(E[X])$, e.g. here $E[X^2] \neq (E[X])^2$

  • Linearity of Expectations:
    • $E[aX + b] = aE[X]+b$
    • Can derive with Expected Value Rule

  • Variance: quantifies the spread of the distribution of a random variable
    • "Distance from the mean $\mu$": Variance ~ $X - \mu$ (which is a random variable in itself)
      • Can calculate $E[X-\mu] = E[x] - \mu = 0$

    • More mathematically sound $\to$ the squared distance from the mean: $\mathbf{var(X) = E[(X - \mu)^2]}$ $\ \ $(note: $var(X) \geq 0$)

    • Calculate using Expected Value Rule $E[g(X)] = \sum\limits_{x}g(x)p_X(x)$ with $g(x) = (x - \mu)^2 \to \mathbf{var(X) = E[g(X)] = \sum\limits_{x}(x-\mu)^2p_X(x)}$
    • Properties:
      • Adding a constant to a random variable does not change the variance: $var(X + b) = var(X) \to$ just shifts PMF left or right
      • Multiplying random variable by constant does the following to variance: $var(aX) = a^2var(X)$

    • Variance can be calculated just with means with handy formula: $\mathbf{var(X) = E[X^2] - (E[X])^2}$
      • Foil out $E[(X - \mu^2)]$ recalling $\mu = E[X]$ to see this is true

  • Standard Deviation: more intuitive measure of spread with same units as random variable
    • Note that variance has units $[m^2]$ if random variable has units $[X]$.
      • Take square root of variance: $\sigma = \sqrt{var(X)}$

Variance of Bernoulli Random Variable:

  • Recall $p_X(1) = p, \ p_X(0) = 1-p$, and $E[X] = p$ for Bernoulli random variable

  • Two ways to calculate variance of Bernoulli:

    1. $var(X) = \sum\limits_{x}(x-\mu)^2p_X(x)$
      • $var(X) = (1-p)^2p + (0 - p)^2(1-p) = p(1-p)$

    2. $var(X) = E[X^2] - (E[X])^2$
      • Note that here, $X^2 = X$ (Bernoulli special case)
      • As such, $var(X) = E[X] - (E[X])^2 = p - p^2 = p(1-p)$

  • So $\mathbf{var(X) = p(1-p)}$ for Bernoulli -- zero variance at p = 0 and p = 1, max variance at p = 0.5

Variance of Uniform Random Variable:

  • Use alternative formula (easiest math):
    • $var(X) = E[X^2] - (E[X])^2 = \frac{1}{n+1}(0^2 + 1^2 + 2^2+...+n^2) - (\frac{n}{2})^2$
    • Use formula for sum in that expression, get $var(X) = \frac{1}{12}n(n+2)$
      • Note that instead of 0 to n, we can say a to b (as long as n = b - a) and variance does not change! Just a PMF shift.
  • As such, make substitution: $\mathbf{var(X) = \frac{1}{12}(b-a)(b-a+2)}$

Conditional PMF and Expectation:

  • Up to now:
    • $p_X(x) = P(X=x)$

    • $\sum\limits_{x}p_X(x) = 1$

    • $E[X] = \sum\limits_{x}xp_X(x)$

    • Expected Value Rule: $E[g(X)] = \sum\limits_{x}g(x)p_X(x)$

    • $var(X) = \sum\limits_{x}(x-E[X])^2p_X(x) = E[X^2] - (E[X])^2$

  • Conditioning on some event A (event A is known to have occured and $P(A) > 0$)
    • $p_{X|A}(x) = P(X = x|A)$

    • $\sum\limits_{x}p_{X|A}(x) = 1$

    • $E[X|A] = \sum\limits_{x}xp_{X|A}(x)$

    • Expected Value Rule: $E[g(X)|A] = \sum\limits_{x}g(x)p_{X|A}(x)$

    • $var(X|A) = \sum\limits_{x}(x-E[X|A])^2p_{X|A}(x) = E[X^2|A] - (E[X|A])^2$

Total Expectation Theorem:

  • Recall Total Probability Theorem: $P(B) = P(A_1)P(B|A_1)+...+P(A_n)P(B|A_n)$
  • If $B = \{X = x\}$:
    • $p_X(x) = P(A_1)p_{X|A_1}(x)+...+P(A_n)p_{X|A_n}(x) \to$ Total Probability Theorem translated to PMF notation

  • Multiplying both sides by $\sum\limits_{x}x$, get:
    • $\mathbf{E[X] = P(A_1)E[X|A_1]+...+P(A_n)E[X|A_n]}$
      • Allows dividing PMF into sections $A_1, A_2,...,A_n$ and "divide and conquer" to find $E[X]$

Conditioning a Geometric Random Variable:

  • Recall Geometric Random Variable properties:
    • $P(H) = p$
    • Random variable $X$: number of trials until first heads
  • "Memorylessness": number of remaining tosses, conditioned on T in first toss, is still geometric with parameter $p$
    • $p_{X-n|X>n}(k) = P(X-n=k|X>n) = P(T_2T_3T_4....H_{k+1}|T_1) = P(T_2T_3T_4....H_{k+1}) = p_X(k)$

    • Bottom line: $\mathbf{p_X(k) = p_{X-n|X>n}(k)}$, and as such, $\mathbf{E[X-n|X>n] = E[X]}$

  • Useful for calculating expectation of geometric distribution:
    • $E[X] = \sum\limits_{k=1}^\infty k p_X(k) = \sum\limits_{k=1}^\infty k (1-p)^{k-1}p$ is hard to compute!

    • Instead: $E[X] = 1 + E[X-1]$ (by linearity of expectations)
      • $E[X] = 1 + E[X-1] = 1 + pE[X-1|X=1] + (1-p)E[X-1|X>1]$ by Total Expectation Theorem

      • $E[X] = 1 + 0 + (1-p)E[X]$

    • As such, $\mathbf{E[X] = \frac{1}{p}}$ for Geometric Random Variable X

Multiple Discrete Random Variables and Joint PMFs:

  • Say we have $X:p_X$ and $Y:p_Y$, two individual random variables and their marginal (individual) PMFs
    • Cannot say anything about relationship between $X$ and $Y$ (e.g. $P(X = Y) = ?$)

  • Joint PMF: $p_{X,Y}(x,y) = P(X = x\ and\ Y = y)$
    • Function of 2 arguments $f(x,y)$

  • $\sum\limits_{x}\sum\limits_{y}p_{X,Y}(x,y) = 1$
  • Can deduce marginal (individual) PMFs from joint PMF: add probability rows/columns accordingly
    • $p_X(x) = \sum\limits_{y}p_{X,Y}(x,y)$ and $p_Y(y) = \sum\limits_{x}p_{X,Y}(x,y)$

  • With joint PMF, we see that $P(X = Y)$ is sum of diagonal probabilities x = y!
  • Can have more than 2 variables:
    • $p_{X,Y,Z}(x,y,z) = P(X = x\ and\ Y = y\ and\ Z = z)$

    • $\sum\limits_{x}\sum\limits_{y}\sum\limits_{z}p_{X,Y,Y}(x,y,z) = 1$

    • $p_X(x) = \sum\limits_{y}\sum\limits_{z}p_{X,Y,Y}(x,y,z)$

    • $p_{X,Y}(x,y) = \sum\limits_{z}p_{X,Y,Y}(x,y,z)$

  • Say we have $Z = g(X,Y)$:
    • $PMF = p_Z(z) = P(Z=z) = P(g(x,y) = z) = \sum\limits_{(x,y):g(x,y)=z}p_{X,Y}(x,y)$

  • Expected Value Rule for Joint PMFs: $\mathbf{E[g(X,Y)] = \sum\limits_{x}\sum\limits_{y}g(x,y)p_{X,Y}(x,y)}$

Linearity of Expectations for Multiple Random Variables:

  • $\mathbf{E[X + Y] = E[X] + E[Y]}$
    • Proven by defining $g(x,y) = x + y$ and using Expected Value Rule for Joint PMFs
    • Extends to $n$ random variables

  • Note that $E[2X+3Y-Z] = 2E[X] + 3E[Y] - E[Z]$

Expectation of Binomial Random Variable:

  • Recall Binomial random variable with parameters $n$ and $p$:
    • $X$: number of successes ($P(success) = p)$ and $n$ is number of independent trials

  • Expectation: $E[X] = \sum\limits_{k=0}^{n}kp_X(k) = \sum\limits_{k=0}^{n}k\binom{n}{k}p^k(1-p)^{n-k} \to$ hard to calculate!
    • Break $X$ into $X = X_1 + X_2+...+X_n$, where $X_i = 1$ if $i$'th trial is success, $X_i = 0$ otherwise

    • Now, $E[X] = E[X_1]+...+E[X_n]$ by linearity of expecations

    • Each indicator variable $X_i$ is Bernoulli variable with $E[X_i] = p$

  • Therefore, $\mathbf{E[X] = np}$ for Binomial random variable

Conditional PMFs Based on Other Random Variables: $A = \{Y=y\}$

  • Already seen conditioning PMF of random variable $X$ on event A: $p_{X|A}(x|A) = P(X=x|A)$

  • What if A is event that some other random variable takes a value? $\to A = \{Y=y\}$
    • $p_{X|Y}(x|y) = P(X=x|Y=y) = \frac{P(X=x\ \cap\ Y=y)}{P(Y=y)}$

  • This leads to definition of conditioning on another random variable: $\mathbf{p_{X|Y}(x|y) = \frac{p_{X|Y}(x,y)}{p_Y(y)}}$, defined for $y$ where $p_Y(y) > 0$
    • Note that $\sum\limits_{x}P_{X|Y}(x|y) = 1$

    • Also leads to $p_{X|Y}(x,y) = p_Y(y)p_{X|Y}(x|y)$ and similarly, $p_{X|Y}(x,y) = p_X(x)p_{Y|X}(y|x)$

  • Similar forms for more than 2 random variables (can condition on one or more random variables)

  • Previously:

    • Multiplication Rule: $P(A \cap B \cap C) = P(A)P(B|A)P(C|A \cap B)$

    • Conditional Expecation: $E[X|A] = \sum\limits_{x}xp_{X|A}(x)$

    • Expected Value Rule: $E[g(X)|A] = \sum\limits_{x}g(x)p_{X|A}(x)$

    • Total Probability Theorem: $p_X(x) = P(A_1)p_{X|A_1}(x)+...+P(A_n)p_{X|A_n}(x)$
      • Given that $\Omega$ partitioned into $A_1, A_2,...,A_n$

    • Total Expectation Theorem: $E[X] = \sum\limits_{i}P(A_i)E[X|A_i]$
      • Given that $\Omega$ partitioned into $A_1, A_2,...,A_n$

  • Conditioning on $A = \{Y=y\}$

    • Multiplication Rule: if $A = \{X=x\}$, $B = \{Y=y\}$, $C = \{Z=z\} \to p_{X,Y,Z}(x,y,z) = p_X(x) p_{Y|X}(y|x)p_{Z|X,Y}(z|x,y)$

    • Conditional Expectation: $E[X|Y=y] = \sum\limits_{x}xp_{X|Y}(x,y)$

    • Expected Value Rule: $E[g(X)|Y=y] = \sum\limits_{x}g(x)p_{X|Y}(x|y)$

    • Total Probability Theorem: $p_X(x) = \sum\limits_{y}p_Y(y)p_{X|Y}(x|y)$
      • Given that $\Omega$ partitioned into $A_1, A_2,...,A_n$, with $Y = \{y_1,...,y_n\}$ and $A_i = \{Y=y_i\}$

    • Total Expectation Theorem: $E[X] = \sum\limits_{y}p_Y(y)E[X|Y=y]$
      • Given that $\Omega$ partitioned into $A_1, A_2,...,A_n$, with $Y = \{y_1,...,y_n\}$ and $A_i = \{Y=y_i\}$

Independence of Random Variables:

  • Recall independence of 2 events: $P(A \cap B) = P(A)P(B)$ and $P(A|B) = P(A)$

  • Independence of a Random Variable and some event A: $P(X = x \cap A) = P(X=x)P(A)$ for all x
    • PMF of X does not change given A: $p_{X|A}(x) = p_X(x)$ for all x, and by symmetry $P(A|X=x) = P(A)$ for all x

  • For independence of Two Random Variables: $P(X = x \cap Y = y) = P(X=x) P(Y=y)$ for all $x, y$
    • Formally, $p_{X,Y}(x,y) = p_X(x)p_Y(y)$ for all $x,y$

    • Can say $p_{X|Y}(x|y) = p_X(x)$ and $p_{Y|X}(y|x) = p_Y(y)$

    • Extends to $n$ random variables

Independence and Expectation:

  • If $X$ and $Y$ are independent: $E[XY] = E[X]E[Y]$

  • Additionally, if $g(X)$ and $h(Y)$ are also independent: $E[g(X)h(Y)] = E[g(X)]E[h(Y)]$
    • Can prove these with Expected Value Rule for Joint PMFs

Independence and Variance:

  • Recall that $var(aX) = a^2var(X)$ and $var(X+a) = var(X)$, and these are always true

  • If $X$ and $Y$ independent, $\mathbf{var(X+Y) = var(X) + var(Y)} \to$ NOT true generally

Variance of Binomial Random Variable: $X$: number of successes in $n$ independant trials with $P(success) = p$

  • Break $X$ into $X = X_1 + X_2+...+X_n$, where $X_i = 1$ if $i$'th trial is success, $X_i = 0$ otherwise

  • Now, $var(X) = var(X_1)+...+var(X_n) = n*var(X_i)$ by independence of indicator variables $X_i$

  • Each indicator variable $X_i$ is Bernoulli variable with $var(X_i) = p(1=p)$

  • As such, $\mathbf{var(X) = np(1-p)}$

#1: PMF of a function of a random variable:

  1. Because $\sum\limits_{x}p_X(x) = 1$, can solve for $a$:

    • $2\frac{9}{a} + 2\frac{4}{a} + 2\frac{1}{a} = 1 \to a = 28$

  2. Z is new random variable $\to p_Z(z) = ?$

    • Possible vales of Z: $Z = \{9, 4, 1\}$

    • Z takes on value of 1 when X = 1 and -1: $\{Z = 1\} = \{X = 1\} \cup \{X = -1\} = \frac{1}{a} + \frac{1}{a} = \frac{2}{a}$
    • Similar argument for 4 and 9: $p_Z(4) = \frac{8}{a}$ and $p_Z(9) = \frac{18}{a}$ where again, $a = 28$
    • Specify that $p_Z(z) = 0$ otherwise

#2: Sampling people on buses. Four buses carrying 148 job-seeking MIT students arrive at a job convention. The buses carry 40, 33, 25, and 50 students, respectively. One of the students is randomly selected. Let X denote the number of students that were on the bus carrying this randomly selected student. One of the 4 bus drivers is also randomly selected. Let Y denote the number of students on his bus.

1. Which of E[X] or E[Y] do you think is larger? Give your reasoning in words.

2. Compute E[X] and E[Y].

  1. Take to extreme: 2 busses, one with 1, one with 1000. E[X] here is greater, as overwhelmingly likely (1000/1001) to select student from large bus. Same applies here.

    • Find PMF of X: $p_X(40) = \frac{40}{148}$, etc $\to$ PMF favored towards larger values
    • Find PMF of Y: $p_Y(40) = \frac{1}{4}$, etc. $\to$ PMF equal at all Y = y at 1/4
    • $E[X] = \sum\limits_{x}xp_X(x) = 39$
    • $E[Y] = \sum\limits_{y}yp_Y(y) = 37$

**#3:

  1. $X: $ non negative integer values
    • $E[X] = \sum\limits_{k = -\infty}^\infty k p_X(k) = \sum\limits_{k = 1}^\infty k p_X(k)$
    • $\sum\limits_{k = 1}^\infty P(X \geq k) = \sum\limits_{k = 1}^\infty \sum\limits_{i = k}^\infty p_X(i) = \sum\limits_{i = 1}^\infty \sum\limits_{k = 1}^i p_X(i) \to$ sneaky exchange of summations!
  • $\sum\limits_{i = 1}^\infty \sum\limits_{k = 1}^i p_X(i) = \sum\limits_{i = 1}^\infty i p_X(i) \to$ this is $E[X]$

  1. $b \geq a \geq 0 \to$ a and b are integers, so the above formula applies
    • $P(Y \geq k) = ?$
      • When $k \leq a \to P(Y \geq k) = 1$
      • When $k > b \to P(Y \geq k) = 0$
      • When $a < k \leq b \to P(Y \geq k) = \frac{b - k + 1}{b - a + 1}$

    • $E[Y] = \sum\limits_{k = 1}^\infty P(Y \geq k) = \sum\limits_{k = 1}^a 1 + \sum\limits_{k = a+1}^b \frac{b - k + 1}{b - a + 1} + \sum\limits_{k = b+1}^\infty 0 = a + \frac{b - a}{2} = \frac{b + a}{2}$
      • This is what we expect for a uniform random variable (expectation is the midpoint)

#4: Coupon collector problem. A particular professor is known for his arbitrary grading policies. Each paper receives a grade from the set {A,A−,B+,B,B−,C+}, with equal probability, independently of other papers. How many papers do you expect to hand in before you receive each possible grade at least once?

  • Set of $k$ "coupons", where in this case, $k = 6$ (number of grades)

  • Define $Y_i$ as number of papers until i'th new grade

    • E.g. A, A-, A, B gives $Y_1 = 1, \ Y_2 = 2, \ Y_3 = 4$. We want $Y_6$

  • Define $X_i = Y_{i+1} - Y_i \to X_i$ is the number of papers you need after $Y_i$ to get the next new grade

    • Now, $Y_6 = \sum\limits_{i=0}^5 X_i$

  • Recall geometric distribution models waiting times or number of trials until success is seen

    • So if $p_i$ is probability of drawing new grade after i grades are seen, $X_i = Geo(p_i) = Geo\frac{6-i}{6}$
    • Since for geometric distribution, $E[X] = \frac{1}{p}$, here $E[X_i] = \frac{6}{6-i}$

  • Now calculate $E[Y_6] = E[\sum\limits_{i=0}^5 X_i] = \sum\limits_{i=0}^5 E[X_i]$ by linearity of expectations

    • $E[Y_6] = 6 \sum\limits_{i = 0}^5 \frac{1}{6-i} = 6 \sum\limits_{i = 1}^6 \frac{1}{i} = 14.7$

  • Generally: $E[Y_k] = k \sum\limits_{i = 1}^{k-1} \frac{1}{i} \approx k\ln k \to$ takes about $k \ln k$ independent trials to collect all k coupons when drawn with replacement

#5: Conditioning example:

  • Here, we condition a geometric distribution on another independent geometric distribution and arrive at a uniform PMF (dependent only on the sum of the outcomes of the distributions). To make this intuitive, imagine two friends flipping coins. The first takes only 1 flip to get heads (X = 1), and the second takes 9 flips (Y = 9). This means n = 10. Because X = 1 is relatively likely and Y = 9 is relatively unlikely, they "cancel" the effect of one another. So as long as the two friends sum to 10, the probability that it took the first friend i flips to get heads is uniform on $i = 1, 2,..., n-1$

  • Recall for geometric distribution (number of indpendent trials until success occurs): $p_X(k) = (1 - p)^{k - 1}p$

  • Conditioning the expression: $P(X = i | X + Y = n) = \frac{P(X = i) \cap P(X + Y = n)}{P(X + Y = n)}$

  • Numerator: $P(X = i) \cap P(X + Y = n) = P(X = i) \cap P(Y = n - i) = P(X=i)* P(Y = n-i)$ because X and Y independent

    • $P(X=i)* P(Y = n-i) = (1 - p)^{i - 1}p * (1 - p)^{n - i - 1}p = (1 - p)^{n - 2}p^2$

  • Denominator: use Total Probability Theorem $\to P(X + Y = n) = \sum\limits_{k = 1}^{n - 1}P(X = k)P(X + Y = n | X = k) = \sum\limits_{k = 1}^{n - 1}(1 - p)^{k - 1}p P(Y = n - k) = \sum\limits_{k = 1}^{n - 1}(1 - p)^{k - 1}p (1 - p)^{n - k - 1}p = \sum\limits_{k = 1}^{n - 1}(1 - p)^{n - 2}p^2$

    • Note: could remove conditioning because X and Y are independent

    • Because sum does not depend on $k$, just making copies: $\sum\limits_{k = 1}^{n - 1}(1 - p)^{n - 2}p^2 = (n-1)(1 - p)^{n - 2}p^2$

  • Finally, $P(X = i | X + Y = n) = \frac{1}{n-1}$ cancelling everything out
