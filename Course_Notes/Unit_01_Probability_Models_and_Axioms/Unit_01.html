6.041x - Unit 1: Probability Models and Axioms¶

Notes by Leo Robinovitch¶

Core Concepts:¶

  • Sample space $\mathbf{\Omega}$ contains list (set) of all possible outcomes
    • Mutually exhaustive
    • Collectively exhaustive
    • At the right granularity
  • Event: a subset of the sample space
  • Fundamental Axioms:
    1. Nonnegativity: $P(A) >= 0$
    2. Normalization: $P(\Omega) = 1$
    3. Finite (Countable) Additivity: If $A \cap B = \emptyset$, then $P(A \cup B) = P(A) + P(B)$
      • Generally: $A \cap B \neq \emptyset$, then $P(A \cup B) = P(A) + P(B) - P(A \cap B)$
      • NOTE: additivity only holds for countable (i.e. sequencable) sequences of events (works for area, but not unit square, real line, etc.)
  • Consequences:
    • $P(A) <= 1$
    • $P(\emptyset) = 0$
    • $P(A) + P(A^c) = 1$
    • $P(\{s_1,s_2,...,s_k\}) = P(s_1) + P(s_2) + ... + P(s_k)$
    • If $A \subset B$, then $P(A) <= P(B)$
    • $P(A \cup B \cup C) = P(A) + P(A^c \cap B) + P(A^c \cap B^c \cap C)$
  • Discrete Uniform Law: if $\Omega$ consists of n equally likely elements, and event A consists of k elements. then: $\mathbf{P(A) = k * \frac{1}{n}}$
  • For a continuous sample space, Probability = Area (Uniform Probability Law)
  • Probability calculations:
    1. Specify sample space
    2. Specify probability law
    3. Identify event of interest
    4. Calculate outcome
  • Probability can be interpreted as:
    • Frequencies
    • Descriptions of beliefs
    • Betting preferences

Mathematical Background:¶

Sets:¶

  • A collection of distinct elements
    • e.g. {a, b, c, d} (finite set); Real numbers (infinite set)
  • $x \in S$ --> x is an element of S (and v.v., $x \notin S$)
  • Specifying a set: {$x \in S: \cos{x} > \frac12$}
  • Universal set: $\Omega$
  • Complement of $S$: $S^c$
    • $x \in S^c$ if $x \in \Omega$ AND $x \notin S$
    • $(S^c)^c = S$
  • Empty set: $\Omega^c = \emptyset$
  • Subsets: S is a subset of T if $S \subset T$ (if $x \in S$, then $x \in T$)
    • S can be smaller or equal to T
  • Unions: $x \in S \cup T$ implies $x \in S$ or $x \in T$
  • Intersections: $x \in S \cap T$ implies $x \in S$ and $x \in T$
  • Infinite collection of sets: $S_n,\ n = 1,2,3...$
    • $x \in \bigcup\limits_{n} S_n$ if $x \in S_n$ for some n
    • $x \in \bigcap\limits_{n} S_n$ if $x \in S_n$ for all n
  • Distributive set properties:
    • $S \cap (T \cup U) = (S \cap T) \cup (S \cap U)$
    • $S \cup (T \cap U) = (S \cup T) \cap (S \cup U)$

De Morgan's Laws:¶

  • Allow transfer between unions and intersections:
  1. $(S \cap T)^c = S^c \cup T^c$
    • Or formally, $(\bigcap\limits_{n}S_n)^c = \bigcup\limits_{n}S_n^c$
  1. $(S \cup T)^c = S^c \cap T^c$
    • Or formally, $(\bigcup\limits_{n}S_n)^c = \bigcap\limits_{n}S_n^c$

Sequences and their Limits:¶

  • Any kind of set possible, e.g. $S = \mathbb{R}$ (all real numbers), $S = \mathbb{R^n}$
  • Sequence: a function that associates an element of a set S for any natural number i
    • Evaluate $f(i)$, get $a_i$ (i'th entry in sequence)
  • Example sequences: $a_1, a_2, a_3,...$, or $a_i = \{a_i\}$ where $i \in \mathbb{N} = \{1, 2, 3,...\}$ and $a_i \in S$
  • Typically care if sequence converges to certain number a: $\lim\limits_{i \to \infty}a_i = a$

When Does a Sequence Converge?¶

  • If $a_i < a_{i+1}$, then either sequence converges to $\infty$ or some real number a
  • If $|a_i - a| <= b_i$ for all i and $b_i \to 0$, then $a_i \to a$
    • "Sandwich argument"

Infinite Series:¶

  • Defined as $\sum\limits_{i=1}^{\infty}a_i = \lim\limits_{n \to \infty} \sum\limits_{i=1}^{n}a_i$ --provided the limit exists
    • If $a_i > 0$ for all $a_i$ --> limit exists
    • If $a_i$ terms not all same sign:
      • Limit does not exist
      • Limit may exist but depend on order of summation
    • Limit exists and independant of order of summation if $\sum\limits_{i=1}^{\infty}|a_i| < \infty$

Geometric Series:¶

  • $S = \sum\limits_{i=0}^{\infty}\alpha^i = 1 + \alpha + \alpha^2 + ... = \frac{1}{1-\alpha}$ if $|\alpha| < 1$
    • Proof: $S = 1 + \sum\limits_{i=1}^{\infty}\alpha^i = 1 + \alpha \sum\limits_{i=0}^{\infty}\alpha^i = 1 + \alpha S \to S (1 - \alpha) = 1 \to S = \frac{1}{1 - \alpha}$ (assuming S < $\infty$)

Order of Summation in a Series:¶

  • If $\sum|a_{ij}| < \infty$, then order doesn't matter: $\sum\limits_{i=1}^{\infty}\sum\limits_{j=1}^{\infty}a_{ij} = \sum\limits_{j=1}^{\infty}\sum\limits_{i=1}^{\infty}a_{ij}$
    • Otherwise, somehow it does?!

Countable and Uncountable Sets:¶

  • Countable sets: each element of set can be matched 1:1 with positive integers
    • $\{a_1, a_2, a_3,...\} = \Omega$
    • Examples of countable sets:
      • Positive integers
      • All integers: 0, 1, -1, 2, -2,...
      • Pairs of positive integers
      • Rational numbers $q$, with $0 < q < 1$
  • Uncountable sets: continuous things
    • Examples of uncountable sets:
      • The interval [0,1]
      • The reals, a plane, etc.
  • Why the reals are uncountable --> Cantor's diagonalization argument

#7 Let the sample space be the set of positive integers and suppose that P(n)=1/2n, for n=1,2,. Find the probability of the set {3,6,9,}, that is, of the set of of positive integers that are multiples of 3.

  • Countable additivity: $\frac{1}{2^3} + \frac{1}{2^6} + \frac{1}{2^9} + ... = \alpha + \alpha^2 + \alpha^3 + ... = \frac{\alpha}{1-\alpha} = \frac{\frac18}{1 - \frac18} = \frac17$

#8 Let the sample space be the set of positive integers. Is it possible to have a uniform" probability law, that is, a probability law that assigns the same probability c to each positive integer?

  • No -- if c=0, $P(\Omega) = 1 = 0$. If c > 0, $P(\Omega) > 1$, which contradicts normalization.

#1 Give a mathematical derivation of the formula:
$\mathbf{P((A \cap B^c) \cup (A^c \cap B)) = P(A) + P(B) - 2P(A \cap B)}$

  • Define the following:
    • $A = A$
    • $B = B$
    • $C = A \cap B^c$
    • $D = A^c \cap B$
    • $E = A \cap B$
  • From additivitiy:
    • $P(C \cup D) = P(C) + P(D)$
    • $P(A) = P(C) + P(E)$
    • $P(B) = P(D) + P(E)$
  • Therefore: $P(C \cup D) = P(C) + P(E) + P(D) + P(E) - 2P(E) = P(A) + P(B) - 2P(E)$, which proves the formula

#2 Out of the students in a class, 60% are geniuses, 70% love chocolate, and 40% fall into both categories. Determine the probability that a randomly selected student is neither a genius nor a chocolate lover.

  • We know:
    • $P(G) = 0.6$
    • $P(C) = 0.7$
    • $P(G \cap C) = 0.4$
  • From this:
    • $P(only\ G) = 0.2$
    • $P(only\ C) = 0.3$
    • $P(G,\ C,\ or\ G \cap C) = 0.2 + 0.3 + 0.4 = 0.9$
  • Therefore, $P(neither\ G\ nor\ C) = P(\Omega) - 0.9 = 0.1$

#4 Bonferroni's inequality:
A: Prove that for any two events A1 and A2, we have $\mathbf{P(A_1 \cap A_2) >= P(A_1) + P(A_2) -1}$

  • $P((A_1 \cap A_2)^c) = P(A_1^c \cup A_2^c)$ from De Morgan's laws
  • $P(A_1^c \cup A_2^c) <= P(A_1^c) + P(A_2^c)$ from union bound
  • Finally, given $P(A^c) = 1 - P(A)$, then $1 - P(A_1 \cap A_2) <= 1 - P(A_1) + 1 - P(A_2)$; rearrange and get proof

B: Generalize to the case of n events A1,A2,,An, by showing that $\mathbf{P(A_1 \cap A_2 ... \cap A_n) >= P(A_1) + P(A_2) + ... + P(A_n) - (n-1)}$

  • Very similar argument using De Morgan's laws for n sets
